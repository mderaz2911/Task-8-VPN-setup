## Objective
To understand how VPN works and how it protects privacy.

## VPN Used
ProtonVPN (Free Version)

## Steps Performed
1. Created ProtonVPN account
2. Installed ProtonVPN
3. Connected to VPN server
4. Checked IP on whatismyipaddress.com
5. Verified IP address changed
6. Tested browsing with VPN ON and OFF

## Observations
- IP address changed after connecting VPN
- Internet speed slightly reduced
- Connection was secure and encrypted

## Benefits of VPN
- Hides real IP address
- Encrypts internet traffic
- Protects privacy
- Safe on public WiFi

## Limitations
- Slower internet speed
- Free version has limited servers
- Not 100% anonymous

## Conclusion
VPN improves online privacy and security by encrypting data and hiding user identity.
